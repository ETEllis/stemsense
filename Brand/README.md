# StemSense identity

The mark turns the product mechanism into one compact symbol: two opposing stems curve around a fixed media timeline and meet at a single scrub cursor. It reads as an abstract **S** without becoming a literal monogram.

## Production assets

- `StemSense-mark.svg` — canonical square symbol on obsidian
- `StemSense-lockup-dark.svg` — primary horizontal lockup on obsidian
- `StemSense-lockup-light.svg` — transparent lockup for light surfaces
- `StemSense-app-icon.png` — 1024 px flattened iOS source icon
- `Generated/StemSense-generated-master.png` — selected image-generation master
- `Explorations/` — the three retained concept generations
- `GENERATION_NOTES.md` — selected prompt and refinement provenance

## Palette

| Token | Value | Use |
| --- | --- | --- |
| Obsidian | `#090B0A` | Primary field and dark text |
| Sense | `#DFFF5F` | Stems and active state |
| Cursor | `#F5F7EE` | Fixed point and dark-surface type |

Keep the mark flat, preserve generous clear space, and never add an outline, glow, bevel, or literal earbud silhouette.
